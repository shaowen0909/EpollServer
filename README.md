# EpollServer

#### Environment

___

gcc (Ubuntu 9.3.0-17ubuntu1~20.04) 9.3.0

cmake version 3.16.3

#### e.g.

---

##### 1. Running the program:

```shell
# Make sure you're in the Git repository.
mkdir build
cd build
cmake ../
make
./EpollServer 
```

##### 2. open another terminal and input:

```shell
telnet localhost 6666 
```
